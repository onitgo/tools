Transcript :

Draw a Circle : Start by drawing a circle. This will be the face.

Add Eyes : Draw two small circles for the eyes inside the big circle.

Draw Eyebrows : Above the eyes, draw two slanted lines for angry eyebrows.

Make a Mouth : Draw a small horizontal line for the mouth and add a downturned curve underneath for a frown.

Color : Color the face yellow and the eyes and mouth black.
